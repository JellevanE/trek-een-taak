# Buildings & Environment Sprites Reference

Guide for creating pixel art buildings, structures, and environmental sprites in the medieval fantasy retro style.

## Style Principles

**Size Guidelines:**
- Props (barrels, crates): 16-32 pixels
- Small structures (tents, stalls): 32-64 pixels
- Medium buildings (houses, shops): 48-96 pixels
- Large structures (castles, towers): 64-128+ pixels

**Architectural Design:**
- Medieval European fantasy aesthetic
- Exaggerated proportions for charm
- Vertical emphasis (tall towers, peaked roofs)
- Clear functional purpose from silhouette
- Details readable at distance

## Color Usage

**Building Materials:**
- Stone: Various purple-gray tones
- Wood: Warm browns within purple palette
- Roofs: Blue-purple tiles or thatch
- Windows: Bright accents (yellows for lit windows)
- Doors: Dark purple-brown

**Lighting:**
- Top-down or 3/4 lighting typical
- Shadows cast downward or to side
- Windows can glow (light source)
- Consistent light direction across level

## Technical Considerations

**Perspective:**
- Straight-on, isometric, or 3/4 view
- Stay consistent within same scene
- Buildings should feel grounded

**Modularity:**
- Consider tile-based construction
- Walls, roofs, corners as separate elements
- Allows flexible building composition
- Power-of-2 tiles when possible (16x16, 32x32)

**Details:**
- Windows: Use depth (dark interior, frame)
- Doors: Show entrance clearly
- Texture: Brick/stone patterns subtle
- Flags/banners: Add character
- Weathering: Makes structures feel lived-in

## Props & Objects

**Common Props:**
- Barrels, crates, chests (purple-brown wood)
- Torches, lanterns (bright accent glow)
- Signs, banners (add color variety)
- Fences, walls, gates
- Vegetation (trees, bushes in purple-green)

**Furniture/Interior:**
- Tables, chairs, beds
- Weapons racks, armor stands  
- Bookshelves, storage
- Keep style consistent with architecture

## Technical Specs

- Transparent background when needed
- Consider layering (background, midground, foreground)
- Larger sprites acceptable for structures
- Can be irregular shapes/sizes
- Export at actual size (no upscaling)
