# Creature Sprites Reference

Guide for creating pixel art creature/monster sprites in the medieval fantasy retro style.

## Style Principles

**Size Guidelines:**
- Small creatures (rats, bats): 12-20 pixels
- Medium creatures (wolves, goblins): 24-40 pixels
- Large creatures (dragons, trolls): 48-80 pixels
- Boss creatures: 80-128 pixels or larger

**Creature Design:**
- Exaggerate key features (horns, claws, wings)
- Strong silhouette more important than details
- Use size and proportions to convey threat level
- Fantasy creatures can break realism for style

## Color Usage

**Creature Types:**
- Dragons: Rich purples and blues (see examples)
- Undead: Desaturated purples, darker tones
- Beasts: Natural tones within purple palette
- Demons: Deeper purples, high contrast
- Magical creatures: Brighter accent colors

**Texture Techniques:**
- Scales: Overlapping curved lines
- Fur: Subtle dithering or hair tufts
- Skin: Smooth gradients with highlights
- Fire/magic: Brighter contrasting colors
- Eyes: Small highlight for life

## Animation Considerations

**Idle:**
- Breathing/pulsing
- Small movements (tail swish, wing flutter)
- Builds anticipation

**Movement:**
- Match creature type (slither, fly, walk, crawl)
- Exaggerate motion for readability
- 4-6 frames typical for walk cycles

**Attack:**
- Clear wind-up and follow-through
- Emphasize threatening features
- Impact frame with effects

## Technical Specs

- Transparent background required
- Flexible dimensions based on creature
- Allow asymmetry (unlike humanoid characters)
- Can extend beyond power-of-2 for unique shapes
- Multiple sprite sizes OK for same creature type
