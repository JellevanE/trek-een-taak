# Items & Objects Sprites Reference

Guide for creating pixel art items, weapons, consumables, and collectibles in the medieval fantasy retro style.

## Style Principles

**Size Guidelines:**
- Small items (coins, gems): 8-12 pixels
- Medium items (potions, keys): 12-20 pixels
- Large items (weapons, armor): 16-32 pixels
- Special items: Up to 48 pixels for emphasis

**Item Design:**
- Instantly recognizable purpose
- Iconic shape/silhouette
- Fantasy embellishment appropriate
- Can be stylized/exaggerated
- Clear at small sizes

## Color Usage

**Item Categories:**
- Weapons: Purple-blue metals, highlights
- Armor: Consistent with character armor
- Potions: Glass with colored liquid (bright accents)
- Gold/treasure: Yellow-gold (accent color)
- Magical items: Brighter purples, glowing effects
- Food: Warm tones, limited use

**Visual Hierarchy:**
- Rarity indicated by color intensity
- Common: Muted tones
- Uncommon: Standard palette
- Rare: Brighter tones
- Legendary: Glowing effects, bright accents

## Item Types

**Weapons:**
- Swords: Purple-blue blade, handle detailing
- Axes: Heavy, angular design
- Bows: Curved, strung or unstrung
- Staffs: Ornate top, magical glow
- Daggers: Small, sharp silhouette

**Armor/Equipment:**
- Helmets: Recognizable type (full, open-face)
- Chest pieces: Plate or mail indication
- Shields: Shape defines type, add emblem
- Boots/gloves: Simple, matches armor set

**Consumables:**
- Potions: Flask shape, liquid color coding
- Food: Stylized, recognizable type
- Scrolls: Rolled parchment, magical glow
- Keys: Ornate, color-coded for doors

**Collectibles:**
- Coins: Gold circle with detail
- Gems: Faceted, colored
- Runes: Carved stone with symbol
- Quest items: Unique, memorable design

## Technical Specs

- Square dimensions preferred (16x16, 32x32)
- Transparent background
- Can have drop shadow for depth
- Icon and inventory versions may differ in size
- Consider animation (sparkle, rotate, hover)

## UI Considerations

**Inventory Icons:**
- Clear at 16x16 or 32x32
- High contrast for readability
- Consistent size within category
- Can have background frame/border

**World Sprites:**
- Slightly larger than inventory (for visibility)
- Can have idle animation
- Glow or sparkle effects
- Shadow beneath for grounding

## Examples Reference

Look at the color palette from example sprites:
- Purple-blue tones for metals (swords, armor)
- Dark purple for wood/leather (bow, staff)
- Cream/off-white for highlights (see castle)
- Bright accents sparingly (magic effects)
- Consistent shading technique across all items
